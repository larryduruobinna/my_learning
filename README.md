# my_learning

Python practice files from my self-study, working toward data analysis
and industrial AI. I'm starting an M.Eng in Applied AI at McGill in
August 2026, coming from an electrical engineering background, so most
of my practice data is themed around electrical systems.

## Structure

- **python-basics/** — loops, functions, dictionaries, error handling,
  file handling, and practice challenges.
- **numpy/** — arrays, indexing, boolean masking, statistics, and
  small EE-flavored projects.
- **pandas/** — reading/writing CSVs, groupby, handling missing data,
  sorting and filtering practice.
- **data/** — practice datasets I made for the exercises (power grids,
  substations, turbines, sensor logs).
